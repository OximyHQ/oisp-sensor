## Quick Start

```bash
# Install
curl -sSL https://sensor.oisp.dev/install.sh | sh

# Start capturing
sudo oisp-sensor record

# Open Web UI
# http://localhost:7777
```

## What It Captures

| Event Type | Description |
|------------|-------------|
| `ai.request` | LLM API requests with model, messages, tools |
| `ai.response` | Responses with tokens, latency, finish reason |
| `agent.tool_call` | Agent tool invocations (MCP, functions) |
| `process.exec` | Process execution with command line |
| `file.open` | File operations |
| `network.connect` | TCP connections |
